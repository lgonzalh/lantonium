PayTollApp

Plataforma integral en la nube para la gestión automatizada de pagos de peaje y recargas electrónicas. Proyecto final de Grado para Tecnologo ADSO Sena 2023-2024

Reseña

PayTollApp es una solución web full-stack diseñada para centralizar y agilizar las operaciones de peaje electrónico. El sistema integra un backend robusto y escalable desarrollado en .NET 8 con una interfaz de usuario limpia e intuitiva. Mediante una arquitectura serverless desplegada en Google Cloud Run y Firebase Hosting, garantiza alta disponibilidad, seguridad en las transacciones y una experiencia fluida para la administración de vehículos, tarjetas y pagos, eliminando fricciones operativas.

Funcionalidades

  • Gestión integral de usuarios, vehículos y tarjetas de peaje vinculadas.
  • Procesamiento de pagos y recargas de saldo en tiempo real.
  • Generación automática de extractos y consultas de historial de movimientos.
  • Arquitectura serverless escalable alojada en Google Cloud Run y Firebase Hosting.
  • Módulo interactivo "Shortcut" para la ejecución y diagnóstico de consultas SQL.
  • Persistencia de datos segura mediante PostgreSQL en Supabase.
