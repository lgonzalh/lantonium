Imbalances

Resena

Imbalances es una solucion de ingenieria financiera en .NET que optimiza la auditoria de holdings mediante el procesamiento acelerado de volumenes de archivos Excel. La arquitectura permite integracion recursiva con OneDrive, validando la reciprocidad entre Cuentas por Cobrar (CxC) y Cuentas por Pagar (CxP). Opera directamente sobre archivos fuente garantizando trazabilidad inmediata sin infraestructura de base de datos adicional.

Funcionalidades

Procesamiento masivo sobre multiples libros Excel en subcarpetas distribuidas.

Auditoria de reciprocidad: identificacion de desequilibrios CxC vs CxP entre entidades del holding.

Validacion automatizada de presencia de reportes criticos para cierre mensual.

Persistencia directa sobre plantillas originales sin dependencia de base de datos.

Panel de disconformidades en tiempo real para toma de decisiones contables.

Pila Tecnologica

.NET (plataforma) — C# (lenguaje) — Microsoft Excel Integration — OneDrive API — Desktop Client.

OFUserControlApp

Resena

OFUserControlApp es una solucion desarrollada en ASP.NET Core 10.0 para la gestion critica de identidades mediante el procesamiento automatizado de archivos Excel de alta volumetria. Implementa Clean Architecture para separacion estricta de intereses, con PostgreSQL sobre Supabase como motor de persistencia cloud-native. El sistema valida usuarios contra el Directorio Activo detectando discrepancias de acceso y estados de actividad en entornos empresariales complejos.

Funcionalidades

Validacion cruzada automatizada entre listados externos y el Directorio Activo.

Arquitectura N-Layer escalable: Domain, Application e Infrastructure.

Monitoreo transaccional en tiempo real mediante SignalR.

Motor de integracion de hojas de calculo para extraccion y exportacion de reportes tecnicos.

Pila Tecnologica

.NET 10 / ASP.NET Core (plataforma) — C# (lenguaje) — Clean Architecture — PostgreSQL — Supabase — SignalR.

PayTollApp

Resena

PayTollApp es una solucion web full-stack disenada para centralizar y agilizar las operaciones de peaje electronico. El sistema integra un backend robusto y escalable desarrollado en .NET 8 con una interfaz limpia e intuitiva. Mediante una arquitectura serverless desplegada en Google Cloud Run y Firebase Hosting, garantiza alta disponibilidad, seguridad en las transacciones y experiencia fluida para la administracion de vehiculos, tarjetas y pagos.

Funcionalidades

Gestion integral de usuarios, vehiculos y tarjetas de peaje vinculadas.

Procesamiento de pagos y recargas de saldo en tiempo real.

Generacion automatica de extractos y consultas de historial de movimientos.

Arquitectura serverless escalable en Google Cloud Run y Firebase Hosting.

Modulo Shortcut para diagnostico y ejecucion de consultas SQL.

Persistencia segura mediante PostgreSQL en Supabase.

Pila Tecnologica

.NET 8 (plataforma) — C# (lenguaje) — PostgreSQL — Supabase — Google Cloud Run — Firebase Hosting.

YT Downloader — Clipper

Resena

YT Downloader - Clipper es una aplicacion de escritorio en Python con interfaz grafica que integra yt-dlp y FFmpeg. Permite descargar videos y audios en alta definicion y extraer segmentos de tiempo especificos, reduciendo el uso de ancho de banda y almacenamiento al descargar unicamente el fragmento de interes.

Funcionalidades

Extraccion precisa de segmentos con tiempos de inicio y fin exactos.

Seleccion multi-formato: Video+Audio, Solo Video o Solo Audio hasta 1080p.

Monitoreo en tiempo real: porcentaje, velocidad (MiB/s), ETA y tamano del archivo.

Integracion con portapapeles del sistema para agilizar la entrada de URLs.

Pila Tecnologica

Python 3 — Tkinter — yt-dlp — FFmpeg — Pillow — PyInstaller.

DocuSheetBot

Resena

DocuSheetBot acelera la gestion documental en dispositivos moviles Android. Procesa imagenes fisicas de documentos, extrae campos de datos mediante tecnicas locales de reconocimiento inteligente y sincroniza la informacion estructurada con hojas de calculo en la nube para auditoria y reporte inmediato en campo.

Funcionalidades

Escaneo y procesamiento inteligente de documentos fisicos.

Sincronizacion automatizada con hojas de calculo en la nube.

Validacion y auditoria de cambios en registros sincronizados.

Diseño optimizado para operacion movil de campo.

LocalTranscriptor

Resena

LocalTranscriptor es una solucion especializada en la conversion de senales acusticas a texto mediante procesamiento estrictamente local en dispositivos Android. Transforma flujos de voz en tiempo real y archivos multimedia preexistentes en registros textuales, sin conexion a internet, garantizando latencia minima y proteccion total de los datos procesados.

Funcionalidades

Transcripcion instantanea con reconocimiento de pausas y segmentos de voz.

Procesamiento de archivos locales en formatos comprimidos y de alta fidelidad.

Almacenamiento persistente en base de datos local para consulta diferida del historial.

Ejecucion robusta en segundo plano para captura sin interrupciones.

Pila Tecnologica

Kotlin — Android SDK — Local On-Device AI models — SQLite.
